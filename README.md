# Boolean-Compressed-Sensing
Data Compression, TU Delft, Q3, Group Project

# Repository Contents

- `main.m`: main script
- `COMP.m`: Combinatorial Orthogonal Matching Pursuit algorithm
- `DD.m`: Definite Defective algorithm
- `MP.m`: Matching Pursuit algorithm
- `nnLASSO.m`: non-negative LASSO algorithm
- `report.pdf`: project report
- `GroupTesting.mat`: data

# Summary

Implemented and compared several sparse recovery algorithms for Boolean compressed sensing in the context of group testing.

# Requirements

- MATLAB R2024b
